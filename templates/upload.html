Загрузка данных

Загрузите файл с данными о клиентах.
Поддерживаемые форматы: CSV (.csv) и Excel (.xlsx, .xls)

Перетащите файл сюда или кликните для выбора

.csv, .xlsx, .xls

Обязательные колонки:
  • user_id — уникальный ID клиента
  • date — дата платежа (YYYY-MM-DD)
  • revenue — сумма платежа
  • status — active или churned
  • last_activity_date — дата последней активности
  • support_ticket_text — текст обращения в поддержку
Дополнительные колонки (опционально):
  • visitors — количество посетителей
  • returns — количество возвратов
  • carts_started — начатые корзины
  • carts_completed — завершённые корзины
  • is_fraud — мошенническая транзакция (true/false)
  • interest_income — процентный доход
  • interest_expense — процентный расход
  • cogs — себестоимость
  • avg_inventory — средний запас
  • staff_count — количество сотрудников
  • payment_status — статус платежа (failed/success)
